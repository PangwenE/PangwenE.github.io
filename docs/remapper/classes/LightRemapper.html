normalizeWithChanges

  • normalizeWithChanges(map: number[][]): LightRemapper
    • Defined in light_remapper.ts:188

    Normalizes a sequence of lightIDs to a sequence of: 1, 2, 3, 4, 5... etc. Accounts for differences changing at different points.

    Parameters

    • map: number[][]

      [[start, step], [start, step]...]

      start - The point at which the differences change.

      step - The new differences.

      If the sequence goes: 1, 3, 5, 6, 7, the differences change from 2 to 1 at the third number. So map would look like: [[1, 2], [3, 1]]

    Returns LightRemapper
