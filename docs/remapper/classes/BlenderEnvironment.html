assignObjects

  • assignObjects(tracks: string | string[], scale?: Vec3, anchor?: Vec3, disappearWhenAbsent?: boolean): void
    • Defined in environment.ts:264

    Assign pre-existing tracks to be animated with this environment. For example if the current data track is "shore" and you assign an object with track "cloud", there needs to be data for the track "shore_cloud". In your shore model if you set the second material name to shore_cloud, you can move represent the cloud's transform by that object.

    Parameters

    • tracks: string | string[]

      Can be a single track or array of tracks.

    • scale: Vec3 = undefined
    • anchor: Vec3 = undefined
    • disappearWhenAbsent: boolean = true

      Determine whether to make this object disappear when no data for it is present in an environment.

    Returns void
